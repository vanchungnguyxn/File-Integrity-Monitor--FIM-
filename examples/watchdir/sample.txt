This is a sample file for testing the File Integrity Monitor.

You can modify this file to see how FIM detects changes.

Some sample content:
- Configuration files
- Log files
- Important documents
- System files

The FIM tool will detect when this file is:
- Modified (content changes)
- Deleted
- Renamed/moved

Try running:
1. fim init --path examples/watchdir --baseline baseline.json
2. fim watch --path examples/watchdir --baseline baseline.json --events events.json
3. Edit this file while monitoring is running
4. Stop monitoring with Ctrl+C
5. fim report --events events.json --out report.html
